# VicioPerpetuo

Este es un proyecto de una tienda virtual de la editorial peruana "Vicio Perpetuo Vicio Perfecto". La tienda acepta tres métodos de pago: Yape, Plin y Bitcoin.

## Tecnologías Utilizadas

El proyecto está desarrollado utilizando las siguientes tecnologías:

- Ruby on Rails: Un framework de desarrollo web para el lenguaje de programación Ruby.
- JavaScript: Un lenguaje de programación ampliamente utilizado para crear interactividad en páginas web.
- HTML5: La última versión de HTML, el lenguaje de marcado utilizado para crear páginas web.
- CSS3: El último estándar de CSS, el lenguaje utilizado para estilizar páginas web.

## Instalación

Para ejecutar este proyecto en tu máquina local, sigue estos pasos:

1. Clona este repositorio en tu máquina local utilizando Git:

```
git clone https://github.com/fabohax/vicioperpetuo.git
```

2. Instala las dependencias del proyecto utilizando Bundler. Asegúrate de tener Ruby y Bundler instalados en tu sistema:

```
bundle install
```

3. Configura la base de datos. Ejecuta las migraciones de la base de datos para crear las tablas necesarias:

```
rails db:migrate
```

4. Inicia el servidor Rails:

```
rails server
```

5. Abre tu navegador web y visita [http://localhost:3000](http://localhost:3000) para ver la aplicación en funcionamiento.
